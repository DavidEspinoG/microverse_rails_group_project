### Key Features <a id="key-features"></a>

- Stores all the ingredients of a recipe
- Shows a list of public recipes
- Generates a shopping list for each user 
- Stores foods for each user

## 💻 Getting Started <a id="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

> Ruby 3.2 installed in your machine
### Setup

Clone this repository to your desired folder:

> git clone https://github.com/DavidEspinoG/microverse_rails_group_project.git

### Install

Install this project with:

bundle install

### Usage

rails s and open http://localhost:3000


### Run tests

To run tests, run the following command:

rspec
